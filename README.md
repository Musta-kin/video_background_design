🎥 Background Video Web Design
A minimalistic and modern webpage design that features a fullscreen background video, creating a visually engaging user experience. This design is perfect for landing pages, portfolios, or creative brand websites that want to make a strong first impression.

🧩 Features
1. 🎬 Fullscreen video background
2. 📱 Fully responsive layout (mobile & desktop friendly)
3. 💡 Clean and minimalistic user interface
4. ⚡ Smooth animations and transitions
5. 🖼️ Optimized for performance and fast loading

🛠️ Technologies Used
1. HTML5
2. CSS3 
3. JavaScript 

🚀 How to Use
Clone this repository:
1. git clone https://github.com/Musta-kin/video_background_design.git
2. - Open the project folder:
   - cd video_background_design
3. Run the project by opening index.html in your browser.
